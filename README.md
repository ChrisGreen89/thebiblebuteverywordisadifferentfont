# The Bible, But Every Word Is a Different Font

A typographic crime against humanity.

**Live site:** [thebiblebuteverywordisadifferentfont.chrisgreen.dev](https://thebiblebuteverywordisadifferentfont.chrisgreen.dev)

## What is this?

A single-page web app that displays random Bible verses with each word rendered in a different Google Font. Because God works in mysterious ways, and so does typography.

## Features

- **180+ Google Fonts** - From elegant serifs to Comic Sans to horror fonts
- **31,102 verses** - Random verses from the KJV Bible via [bolls.life API](https://bolls.life/api/)
- **Infinite chaos** - Every refresh shuffles the fonts

## Font Categories

- Clean & Professional (Roboto, Inter, Space Grotesk)
- Cursive & Script (Pacifico, Dancing Script, Great Vibes)
- Display & Fun (Bangers, Luckiest Guy, Freckle Face)
- Horror & Spooky (Creepster, Nosifer, Butcherman)
- Retro & Typewriter (Press Start 2P, VT323, Special Elite)
- Medieval & Fancy (Cinzel, Uncial Antiqua, MedievalSharp)
- Futuristic & Tech (Orbitron, Audiowide, Oxanium)
- Handwritten (Indie Flower, Rock Salt, Crafty Girls)
- And more...

## Running Locally

```bash
# Just open the file
open index.html

# Or serve it
python3 -m http.server 8000
```

## Tech Stack

- HTML/CSS/JS (no frameworks, no build step)
- Google Fonts API
- [bolls.life](https://bolls.life/api/) Bible API
